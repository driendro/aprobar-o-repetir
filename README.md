# Aprobar o Repetir

Juego idle/incremental narrativo en el que sos estudiante de una universidad pública argentina en medio de un ajuste presupuestario. Inspirado en *A Dark Room*. Humor negro. Sátira política. Sin final definido — es un loop, igual que la lucha.

**Jugá en:** https://driendro.github.io/aprobar-o-repetir

## Mecánicas

- Los recursos (apuntes, mate, café, voluntad) se acumulan solos una vez desbloqueados
- El `presupuesto_facu` cae por sí solo con el tiempo
- Las acciones colectivas (asamblea, paro, marcha) frenan temporalmente la caída
- El gobierno hace movimientos que podés aceptar o rechazar — cada decisión tiene consecuencias
- El juego guarda el estado en tu navegador (localStorage)

## Créditos

Hecho con HTML, CSS y JavaScript vainilla. Sin dependencias externas.
